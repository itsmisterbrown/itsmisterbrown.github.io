pico<\/em><\/strong>: an R package that provides a suite of compositional data analysis (CoDA) functions targeted for marker gene (16S\/18S, etc.) microbiome datasets (though really applicable to any n x p<\/em> count matrix).<\/p>\n

Affiliated manuscript: The associated manuscript is in preparation but parts of the package were used for analysis in:<\/p>\n

Brown BP,\u00a0<\/strong>Jaspan HB. Compositional analyses reveal correlations between taxon-level gut bacterial abundance and peripheral T cell marker expression in African infants. Gut microbes. 2019 Jul 28:1-8. PMID31347944<\/p>\n

Package will be available via Bioconductor shortly.<\/p>\n<\/div>\n<\/div><\/div>

microfiltR<\/a>: an R package that aims to identify and mitigate exogenous- and cross-contamination in compositional marker gene (16S\/18S, etc.) surveys.<\/p>\n

Affiliated manuscript: The associated manuscript is in preparation but parts of the package were used for analysis in:<\/p>\n

Brown BP<\/strong>, Wendoh J, Chopera D, Havyarimana E, Jaumdally SZ, Nyangahu DD, Gray C, Martin DP, Varsani A, Jaspan HB. crAssphage abundance and genomic selective pressure correlate with altered bacterial abundance in the fecal microbiota of South African mother-infant dyads. bioRxiv. 2019 Jan 1:582015.<\/p>\n<\/div>\n<\/div><\/div><\/div><\/div><\/div><\/div>

marker gene dataset merging functions<\/a>: a collection of R functions that provide several statistical transformations and standardizations for the merging of marker gene datasets (16S\/18S, etc.) while accurately preserving relative ratios of taxa across datasets. Compatible with phyloseq-class<\/a> datasets.<\/p>\n<\/div>\n<\/div><\/div>
